# AWS RDS MySQL Project

## API

- Built with FastAPI
- Endpoints:
  - `/top-customers`
  - `/monthly-sales`
  - `/products-never-ordered`
  - `/avg-order-country`
  - `/frequent-buyers`

## Setup

3. Install FastAPI: `pip install -r requirements.txt`
4. Change directory: `cd api`
5. Run: `uvicorn db_api:app --reload`
6. Visit docs: `http://34.241.173.68:8000/docs`
